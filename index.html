木易跟打器

木易跟打器是一款支持macOS、Web使用的跨平台打字练习程序，是macOS平台唯一的、可直接通过QQ群载文的跟打器

跨平台

  • Web 在线跟打练习
  • macOS 平台上直接载文
  • 基于PWA的渐进式免安装 Web App
  • Android/iOS 上的微App
马上体验

功能丰富

  • 阅读、发文、跟打、成绩可视化等
  • 编码提示、支持复合指标练习
  • 速度、击键、码长等丰富成绩指标
  • 内置单字+名篇等丰富练习素材
更多功能

安全性

  • 基于浏览器 IndexDB 的数据本地化
  • 无任何数据后台上传
  • 虎码用户设置预设
