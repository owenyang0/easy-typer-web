In version 6.5.5, we added support for ISO week, improved the migration
experience when importing CSV files from PostgreSQL, and allowed users to attach
partitions using symbolic links. Besides, there are several stability
improvements for the Core and SQL engine.

[full release notes](https://github.com/questdb/questdb/releases/tag/6.5.4).
